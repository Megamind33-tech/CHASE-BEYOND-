# Roadmap

This roadmap protects the foundation from uncontrolled expansion.

## Phase 0 - Governance

Status: started.

Includes:

- Agent rules.
- Architecture direction.
- Dependency policy.
- Quality gate.
- UI contract.
- Schema contracts.
- Testing strategy.

## Phase 1 - Web 3D foundation

Goal: prove the basic virtual studio works.

Includes:

- Vite, React, TypeScript app.
- Babylon.js runtime.
- Starter set loading.
- Source manager.
- Video texture on a 3D screen mesh.
- Source replacement.
- Set replacement.
- Diagnostics panel.
- Browser smoke check.

## Phase 2 - OBS output proof

Goal: prove OBS can capture the Beyond output.

Includes:

- OBS Browser Source guide.
- Window capture fallback.
- Recording check.
- Visual proof.
- Known limitations.

## Phase 3 - Operator usability

Goal: make the app usable for a real operator.

Includes:

- Project save and load.
- Better source list.
- Better set library.
- Cleaner diagnostics.
- Better empty states.
- Starter installer plan.

## Phase 4 - Advanced broadcast options

Only after phase 1 and 2 are proven.

Candidates:

- NDI.
- DeckLink.
- GStreamer.
- Direct OBS or libobs integration.
- Electron packaging.
- Production audio routing.

## Phase rule

Do not work on a later phase if the earlier phase is still unstable.
